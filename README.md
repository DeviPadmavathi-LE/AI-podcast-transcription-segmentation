# Automated Podcast Transcription and Topic Segmentation

This project builds an AI system to:
- Transcribe podcast audio using Speech-to-Text
- Segment transcripts into topics
- Generate summaries and keywords

## Tech Stack
- Python
- Speech-to-Text (Whisper / Google ASR)
- NLP (BERT / TextTiling)
- FastAPI
- Streamlit (UI)

## Modules
1. Audio Preprocessing
2. Speech-to-Text
3. Topic Segmentation
4. Summarization
5. API & UI
